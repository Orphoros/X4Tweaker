# X4 Tweaker

## Developer's guide

Install dependencies:

```bash
pip install -r requirements.txt
```

Run in watch mode:

```bash
watchmedo auto-restart -p "*.py" -R briefcase dev
```

Run in dev mode:

```bash
briefcase dev
```

Update dependencies:

```bash
pip freeze > requirements.txt
```
